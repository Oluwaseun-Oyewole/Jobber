### Jobber

#### Project Overview

A job platform

#### Features:

```bash
Post-jobs
Search-for-jobs
Filter-for-jobs
Personal-user-dashboard
Save-jobs
Save-recent-searches
Job-description
```

#### Technologies

```bash
React
NextJS
Tailwind
Typescript
NextAuth
Prisma
PostgreSQL
```
